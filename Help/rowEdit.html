Row Editing

Each row on each sheet of the Excel Data is shown on the Top Left portion of the program window, as a form. Most columns of the row are shown as one line edit controls, with the description shown as a multiline edit, to allow longer descriptions. All the edit controls have labels at the left, with column names.

  • Top of form is a label, with the Opened Excel File Directory and Name, and the sheet name within the file.

  • Sheet Controls - The next line contains sheet control pushbuttons, to go backwards (prev) and forwards (next) through the sheets, a sheet number label, and a dropdown list to select the icon to use for display of the Placemarks of that sheet.

    You should select a different icon for each sheet. The selected icon is not saved in the Excel data, and will have to be reselected each time the program is used. The set of icons are "baked in" the program, and changing them requires changing program code.

  • Row Controls - The bottom of the form contains four pushbuttons, and a row number label. The buttons are to go backwards (prev) and forwards (next) through the rows on the sheet, and Undo and Update row buttons. The Undo and Update row buttons are "greyed out" and disabled unless row data has been changed.

    When row data is changed, the row is locked, and the prev and next row and sheet pushbuttons are "greyed out" and disabled. You will not be able to move between rows or sheets until row changes are Undone or Updated by one of the pushbuttons.

  • Changing row data is done by typing new data into one or more of the edit controls. The program will detect the change, and lock the row, disabling the prev and next pushbuttons, enabling the Undo and Update pushbuttons. Undo will undo all changes to the row, reverting the edit controls for the row to stored values. Update will replace stored data for the row with data from the edit controls.

    There is only one level of Undo. Undo is only possible between changing the row data by editing, and Updating the row. Once changes to a row have been Updated, by pressing the pushbutton, they cannot be Undone.

    After a row is Updated, the Excel data will be flagged as changed, and can be Saved to a file with Excel Save or Excel Save As. If the changed Excel data is not saved before Quiting the program, the Program will ask if you want to save data before Quiting.

  • Lookup - To the right of the Longitude and Latitude edit controls is a Lookup pushbutton. If a row does not have Longitude or Latitude data, but does have enough Street, Town, State, Zip data for a postal address, pressing this button will attempt to use a Google web service to lookup a Longitude and Latitiude for the postal address. If it fails, an error message will be produced. If it succeeds, the Longitude and Latitude edit controls will be filled with the new values, and the row will be flagged as changed, the row will be Locked and require Undo or Update. The computer must be connected to the internet to use this button.
